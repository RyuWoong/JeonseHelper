# Source List

Last checked: 2026-06-23

Official and high-priority sources for JeonseHelper content.

## Core Support

- 전세사기피해자 지원관리시스템: https://jeonse.kgeop.go.kr/
  - Use for victim decision application, decision result, objection, correction, withdrawal, support notices, forms, and related local government contact points.
- 전세사기피해자 지원관리시스템 서식/자료실: https://jeonse.kgeop.go.kr/ntcnYard/tmpltNrecsRoom.do
  - Use for official forms and downloadable materials.
- 서울주거포털 전세사기피해자등 결정신청 안내: https://housing.seoul.go.kr/site/main/content/sh05_070100
  - Use as local-government explanatory reference. Verify against the national system before treating as final.

## Guaranty Institutions

- HUG 전세보증금반환보증 이행청구: https://onestop.khug.or.kr/webView/webBiz/excu/loahExcu001
  - Use for HUG claim reasons and procedure entry.
- HUG 안심전세 App 안내: https://www.khug.or.kr/jeonse/web/s01/s010102.jsp
  - Use for prevention checks and guaranty availability checks.
- HF 전세보증금 반환 청구 절차: https://www.hf.go.kr/ko/sub02/sub02_05_07.do
  - Use for HF claim definitions, guaranty accident criteria, and claim process.

## Source Handling Rules

- Prefer national official sources over private blogs.
- Record `lastVerifiedAt` on every source embedded in JSON content.
- If a source includes policy criteria, treat it as time-sensitive.
- When a local government page is used, verify whether the same rule appears in the national system.
